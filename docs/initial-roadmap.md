# Initial Roadmap

## Vertical slice 1

Goal: prove the architecture and first gameplay loop.

- Account creation
- Character creation
- Dashboard
- Basic stats
- One job action
- One crime action
- Event log
- Cooldown handling
- PostgreSQL persistence
- Redis action lock

## Vertical slice 2

Goal: add delayed state and worker reliability.

- Travel action
- Travel completion job
- Hospitalization state
- Recovery completion job
- Market tick job
- Notification after delayed completion

## Vertical slice 3

Goal: add social and economy hooks.

- Direct messages
- Player shop
- Item listing
- Buy/sell transaction
- Newspaper event feed
- Basic faction creation

## Vertical slice 4

Goal: build first long-term loop.

- Education courses
- Training
- Reputation
- Achievements
- Daily objectives
- Regional market prices
