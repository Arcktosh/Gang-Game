## Public API rules

- Import shared workspace packages through their package name, not deep relative paths.
- Add new shared exports to the package entrypoint and regenerate this memory.
- Keep Next.js route handlers explicit with named HTTP method exports.
- Treat files under `apps/web/src/lib` as app-internal unless promoted through a workspace package.
